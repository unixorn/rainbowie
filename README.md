# rainbowie
My personal theme for oh-my-zsh

## Requirements
Its recommended to use nerd-fonts complete. 

### Used on the image: 
  +DejaVu Sans Mono Nerd Font Complete. 
  +Terminal: termite
  +Colors: Nord

## Installation
Simply drop the .zsh-theme file inside <OMZSH>/custom/thenes/
